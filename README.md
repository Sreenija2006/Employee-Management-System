# Employee Management System

A simple web-based application to manage employee records using Java.

## 🚀 Features
- Add Employee
- View Employee List
- Edit Employee
- Delete Employee (with confirmation)
- Success messages after operations

## 🛠️ Technologies Used
- Java (Servlets)
- JSP
- JDBC
- MySQL
- HTML, CSS

## ▶️ How to Run
1. Import project into Eclipse
2. Configure MySQL database
3. Run on Apache Tomcat server
4. Open in browser: http://localhost:8080/EMS/

## 📌 Project Structure
- Controller → Servlets
- Service → Business logic
- DAO → Database operations
- Model → Data objects
- View → JSP pages
